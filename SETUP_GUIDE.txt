
## ğŸš€ How to Host Your System

### Option 1: Render.com (Recommended - FREE)

1. **Install Gunicorn:**
```powershell
pip install gunicorn
pip freeze > requirements.txt
```

2. **Create `render.yaml`:**
```yaml
services:
  - type: web
    name: loan-predictor
    env: python
    buildCommand: pip install -r requirements.txt
    startCommand: gunicorn app:app
    envVars:
      - key: PYTHON_VERSION
        value: 3.11.0
      - key: GEMINI_API_KEY
        sync: false
```

3. **Push to GitHub:**
```powershell
git push origin main
```

4. **Deploy:**
- Go to https://render.com
- Connect GitHub repo
- Click "Create Web Service"
- **Live in 3-5 minutes!**

## ğŸ› Troubleshooting

### OTP not displaying?
- Check terminal where `python app.py` is running
- OTP prints with decoration for visibility

### Gemini AI not working?
- Verify API key: `echo $env:GEMINI_API_KEY`
- Check https://aistudio.google.com/app/apikey
- Fallback works without key (rule-based)

### Dark mode not saving?
- Enable browser localStorage
- Check browser console for errors

### Database errors?
- Delete `loan_prediction.db`
- Restart app (auto-recreates tables)
